## ✨ 页面特点

### 1. 权限管理
- ✅ 必须登录后才能进入学习资料页面
- ✅ 未登录自动跳转到登录页
- ✅ 上传和下载都需要登录验证

### 2. 页面布局
**左侧分类栏**
- 上方：「上传文件」按钮（蓝色渐变）
- 下方：7个分类按钮
  - 全部资料（默认选中）
  - 考试资料
  - 复习资料
  - 课件讲义
  - 习题作业
  - 实验资料
  - 其他

**右侧资料列表**
- 搜索框 + 排序下拉菜单
- 资料卡片列表
- 每条资料显示：标题、分类、上传人、上传时间、下载按钮

### 3. 上传功能
**上传弹窗**
- 选择文件
- 输入文件名
- 选择分类
- 上传/取消按钮

**支持格式**
- 文档：.pdf, .docx, .doc, .txt
- 压缩包：.zip, .rar, .7z
- 图片：.jpg, .png, .gif
- 其他常见格式

### 4. 资料列表功能
**每条资料显示**
- 标题（大字体、加粗）
- 分类、上传人、上传时间（灰色、小字体）
- 下载按钮（蓝色、可点击）

**交互效果**
- 悬停时卡片向上移动
- 悬停时阴影增强
- 下载按钮悬停时变深

### 5. 搜索和排序
**搜索框**
- 实时搜索资料标题
- 后续对接 Supabase 全文搜索

**排序功能**
- 按上传时间排序
- 按文件名排序

## 🔧 后端对接规划

### Supabase Storage
```javascript
// 文件上传到 Supabase Storage
const { data, error } = await supabase.storage
    .from('study-materials')
    .upload(`${userId}/${fileName}`, file);
```

### Supabase Database
```javascript
// 保存资料信息到数据库
const { data, error } = await supabase
    .from('materials')
    .insert([{
        title: fileName,
        category: category,
        uploader: currentUser.email,
        upload_time: new Date(),
        file_url: downloadUrl,
        user_id: currentUser.id
    }]);
```

### 下载功能
```javascript
// 获取下载链接
const { data } = supabase.storage
    .from('study-materials')
    .getPublicUrl(filePath);
```

## ✨ 主要改进

### 1. 页面背景
**之前**：深蓝色渐变背景（#f5f7fa → #c3cfe2）

- 更清爽、舒适
- 非常柔和的浅色调
- 整体更高级、简洁

### 2. 首页标题重新设计
**"欢迎来到 MY SANDA"**

**字体改进**
- 使用 Segoe UI 现代字体
- 字号：从默认增加到 3.5rem（桌面端）

- 字间距：-1px（紧凑现代感）

- 使用渐变色：#4f46e5 → #7c3aed → #667eea
- 从靛蓝到紫色的现代渐变
- 与淡蓝色背景完美搭配
- 清爽、现代、高级

**排版优化**
- 行高：1.2（紧凑）
- 下边距：1rem
- 整体更有视觉冲击力

### 3. 副标题优化
**"校园生活的交流平台，分享你的故事"**

**样式改进**
- 字号：1.125rem（更大、更清晰）

- 字重：500（中等）
- 字间距：0.5px（现代感）

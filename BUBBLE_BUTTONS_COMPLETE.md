## 🎨 设计特点

### 气泡风格
- **大圆角**：border-radius: 50px，完全圆润
- **柔和渐变**：从浅蓝到稍深的蓝色
- **轻盈阴影**：柔和的蓝色阴影，不生硬
- **流畅动画**：cubic-bezier 缓动函数，更自然

### 颜色方案
- **默认状态**：#e3f2fd → #bbdefb（浅蓝色渐变）
- **悬停状态**：#bbdefb → #90caf9（中蓝色渐变）
- **激活状态**：#64b5f6 → #42a5f5（深蓝色渐变）
- **文字颜色**：白色，带轻微阴影

### 交互效果
- **悬停**：向上浮动 3px，放大 1.05 倍
- **激活**：向上浮动 2px，放大 1.03 倍
- **过渡**：0.3s cubic-bezier 缓动
- **阴影**：随状态变化而增强

## 🔧 技术实现

### CSS 样式
```css
.category-btn {
    background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%);
    color: white;
    border: none;
    box-shadow: 0 4px 15px rgba(100, 181, 246, 0.3);
    transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
    font-weight: 600;
    text-shadow: 0 1px 2px rgba(0, 0, 0, 0.1);
    border-radius: 50px;
}

.category-btn:hover {
    transform: translateY(-3px) scale(1.05);
    box-shadow: 0 6px 20px rgba(100, 181, 246, 0.4);
    background: linear-gradient(135deg, #bbdefb 0%, #90caf9 100%);
}

.category-btn.active {
    background: linear-gradient(135deg, #64b5f6 0%, #42a5f5 100%);
    box-shadow: 0 6px 20px rgba(66, 165, 245, 0.5);
    transform: translateY(-2px) scale(1.03);
}
```

### 关键属性
- **border-radius: 50px**：大圆角，气泡感
- **linear-gradient**：渐变背景，更有层次
- **box-shadow**：柔和阴影，轻盈感
- **transform**：浮动和缩放，动态感
- **cubic-bezier**：自然缓动，流畅感

## 💡 设计理念

### 气泡感
- 大圆角让按钮像泡泡一样圆润
- 柔和的阴影让按钮像浮在空中
- 悬停时的浮动和放大让按钮像泡泡一样轻盈

### 淡蓝色主题
- 浅蓝色渐变让按钮清爽干净
- 白色文字让按钮高级优雅
- 蓝色阴影让按钮和背景融为一体

### 流畅动画
- cubic-bezier 缓动函数让动画更自然
- 0.3s 过渡时间让动画不快不慢
- transform 让动画更流畅

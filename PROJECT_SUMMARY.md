## 🚀 快速开始

### 1. 配置 Supabase（2分钟）
- 访问 https://supabase.com
- 创建新项目
- 获取 API URL 和 anon key
- 关闭邮箱验证

### 2. 更新配置（1分钟）
编辑 `js/app.js` 第 1-2 行：
```javascript
const SUPABASE_URL = 'https://your-project.supabase.co';
const SUPABASE_KEY = 'your-anon-key';
```

### 3. 启动服务器（1分钟）
```bash
# Python
python -m http.server 8000

# 或 Node.js
npm install && npm start

# 或 Windows
双击 start-server.bat
```

### 4. 打开浏览器
访问 `http://localhost:8000`

## 🛠️ 常见问题

### Q: 如何修改网站名称？
A: 编辑 `index.html` 中的 `<title>` 标签和导航栏中的 "MY SANDA" 文本。

### Q: 如何修改颜色主题？
A: 编辑 `css/style.css` 中的渐变色值。

### Q: 如何添加新的分类？
A: 在 `index.html` 中的分类按钮区域添加新按钮。

### Q: 密码重置不工作？
A: 确保已在 Supabase 中禁用邮箱验证。

### Q: 如何部署到线上？
A: 可以部署到 Vercel、Netlify、GitHub Pages 等静态托管服务。
